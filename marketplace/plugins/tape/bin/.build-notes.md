# Bundled binary notes

These binaries are built for **macOS Apple Silicon** (Mach-O arm64). For other platforms, build from source per `README.md` in the plugin root.

## Re-signing after rebuild

When updating these binaries by `cp` from `target/release/`, macOS adds a `com.apple.provenance` xattr that causes the kernel to kill the binary on exec (exit 137). The fix is to ad-hoc re-sign in place:

```sh
cp ../../../target/release/{tape,tape-hook,tape-mcp-wrap} .
codesign --force --sign - tape tape-hook tape-mcp-wrap
```

This is a macOS-only quirk; on other platforms you can simply copy the binaries.
